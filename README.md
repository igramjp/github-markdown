# GitHub Markdown
GitHub Flavored Markdown CSS for Previm.

## Description
Preview with GitHub Flavored Markdown.

## Setup
```
git clone git@github.com:igramjp/github-markdown.git
```

## Usage
Install Previm and add the following to vimrc:
```
let g:previm_open_cmd = 'open -a Firefox'
let g:previm_disable_default_css = 1
let g:previm_custom_css_path = '~/path/to/github-markdown.css'
```
